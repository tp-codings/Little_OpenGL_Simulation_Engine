# Getting Started
1. Clone Repository
`git clone https://github.com/tp-codings/Little_OpenGL_Simulation_Engine.git`
2. Build Executable (e.g. Visual Studio)

# User Manual
Movement:
- W: Forward
- A: Left
- S: Back
- D: Right
- Shift: Down
- Space: Up
- Mouse: Look around
Other:
- F: Open Settings (ImGui)
